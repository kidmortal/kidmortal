Aplicação com React JS

Fiz essa aplicação para resolver alguns problemas que temos na empresa atual.

Utilizamos varios sistemas para gerenciar a empresa, e esses sistemas não possuem conexão direta, porem ambos possuem API.

Então essa aplicação vem para unificar os dados de ambos sistemas, afim de gerar relatorios, fazer alterações ou correções em lote, consultas rapidas, etc.
